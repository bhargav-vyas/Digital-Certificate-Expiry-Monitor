# 🔐 Digital Certificate Expiry Monitor

## 📌 Project Overview
The **Digital Certificate Expiry Monitor** is a Python-based tool that helps users monitor the validity of digital certificates by checking their expiry dates. It reads certificate metadata from JSON files and classifies them as **Expired**, **Expiring Soon**, or **Safe**.

---

## 🎯 Objective
To build a simple working tool that demonstrates the concept of **digital certificates** and their **expiry monitoring**, using sample inputs and clear output.

---

## 🚀 Features
- 📂 Load certificate data from JSON file  
- 🔍 Check certificate expiry status  
- ⚠️ Identify:
  - Expired certificates  
  - Expiring soon (within 30 days)  
  - Valid (safe) certificates  
- 🖥️ Simple GUI using Tkinter  
- 💡 Easy-to-use interface  

---

## 🛠️ Technologies Used
- Python 🐍  
- Tkinter (for GUI)  
- JSON (for data storage)
